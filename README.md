# Line 22

A fast-paced 2D side-scrolling shooter where you pilot a flying tram through chaos.

**2nd Place – Game Jam Project**  
Developed in just **14 hours**

---

## Gameplay

In **Line 22**, you control a flying tram navigating through a dangerous side-scrolling environment.

- Shoot stationary turrets
- Dodge bullets
- Survive until the escape

The game focuses on **tight controls, fast reactions, and arcade-style challenge**.
